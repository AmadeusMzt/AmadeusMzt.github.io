快乐风男——一道算法竞赛入门题

杨泽华 · 2019-12-25 · 次阅读

题目

今天刷题的时候在牛客网看到的，看到一道可以激起大家兴趣的题目，今天就在这里解一下这个题。

链接：https://ac.nowcoder.com/acm/contest/3282/G
来源：牛客网

解题

大家看到了题目以后，会发现可能根本读不懂题目是什么意思（出题人一般都很恶趣味哈哈）。
这是很正常的，程序设计竞赛通常都会设置这样的背景故事（我受够了重庆市ACM今年出的爱喝珍珠奶茶的cyy了），因为编程正是让你将生活中发生的抽象的事情转化为程序，来解决我们实际生活的问题。

所以这也就是第一个考点——如何理解题目的含义，设计一个算法解决问题。

首先玩过《LOL》的人应该很快就能理解lfq这个人操作了，在脑海里可以构建一个你自己玩亚索的场景，就能大概知道题目的背景是什么意思了。

然后看加粗的部分 “如果有多个快乐方案，给出字典序最小的方案。 ”
如果你是一个经验丰富的ACM自动机，那么你很快就能给出这个题目的算法了。

这里，多个方案，能联想到什么呢？

是的，需要用到咱们智能科学与技术导论学习过搜索的算法思想了。

然而在这里我需要考虑是用深度优先搜索还是广度优先搜索还是二分法查找呢？还要涉及到排序，可能会使用剪枝技巧（我当时那节课讲过）。

为了让大家学会学习，我就不作过多的详细介绍，自己去别人的博客查这些算法以及如何用编程语言实，现然后可以自己敲一遍（虽然比赛是可以带任何纸质资料的，但是你想当模板用必须去认真敲一遍然后理解）。

但是这里实际上就是然后给定了一个指定长度序列，在这个序列里面找到最长的上升子序列。举个例子，就是给出了兵线的长度，玩亚索的都懂得E往无前的快乐，前前面有很多小兵，但是你想E得更往前，但是中间有很多小兵，E是有cd的（可以理解为题目中E的花费金币是递增的），那么就肯定要跳过一些小兵。

还有一个是算法是动态规划就是下面代码中以DP(dynamic programing)命名的函数。

在以后要开的集训中我们要讲的两个重点就是搜索和动态规划。

我们今年的C++考试大家应该还记得吧，当时老师让我们把函数声明和头文件分别写在一个.cpp和一个.h里面，然后主程序很简单，直接调用函数即可。实际上C++的STL标准库里包含了各种各样的库函数，我们可以直接调用，比如之前学过的排序sort()就在头文件#include中，sort函数进行排序的时间复杂度为n*log2n，比冒泡之类的排序算法效率要高。

这里我只简单讲一下字典序是什么意思：

所谓字典大家应该都用过，每个单词的排序大家也都很清楚，举个例子，比如admin排在administrator前面，而admin又排在across后面和agree后面，这就是字典序，再例如，第1234页排在第1243页前面，这就是字典序。如果还不是很懂，可以点一下上面橙色字体字典序，我已经把百度百科链接到了。

代码实现

#include <vector>    //这个头文件是用来给数组动态分配空间的
#include <iostream>
#include<algorithm>  //这里面包含了很多有用的函数可以直接调用，比如sort，和reverse
using namespace std;

const int MAXN=100000+5;   //题目要求测试的数据数量，定义为只读，最大值为1e5
int a[MAXN], b[MAXN], d[MAXN];
vector<int> c[MAXN]; //动态分配数组的空间，在算法竞赛中尽量使用数组可以节省程序运行的时间，vector的使用方法也可以百度一下。
int n;

int Search(int num,int low,int high)    //二分法查找（就是曾老师讲过的那个算法，当时是一排数字分成两组排序）
{
    int mid;          //low 是下界，high是上界，mid是中间
    while(low<=high)
    {
        mid=(low+high)/2;
        if(num<b[mid])  low=mid+1;
        else   high=mid-1;
    }
    return low;
}

int DP(int n)     //动态规划
{
    int i,len,pos;
    b[1]=a[1];
    c[1].push_back(1);
    len=1;
    for(i=2;i<=n;i++)
    {
        if(a[i]<b[len])
        {
            len=len+1;
            b[len]=a[i];
            c[len].push_back(i);
        }
        else
        {
            pos=Search(a[i],1,len);
            b[pos]=a[i];
            c[pos].push_back(i);
        }
    }
    return len;
}


int main()
{
    cin>>n;     //输入小兵的数量
    for (int i=1;i<=n;i++)
    {
        cin>>a[i];
    }
    reverse(a+1,a+n+1);
    int m=DP(n);
    cout<<m<<endl;
    int x=n+1;
    for (int i=m;i>=1;--i)
    {
        for (int j=c[i].size()-1;j>=0;--j)
        {
            if (c[i][j]<x)
            {
                d[i]=c[i][j];
                x=c[i][j];
                break;
            }
        }
    }
    reverse(d+1,d+m+1);    //reverse函数是对字符串进行翻转，这里是调用的STL库函数，由#include<algorithm>包含，作为比赛的模板非常常用。
    for (int i=1;i<=m;++i)
    {
        cout<<n+1-d[i];
        if (i!=m) cout<<" ";
        else cout<<endl;
    }
    return 0;
}
