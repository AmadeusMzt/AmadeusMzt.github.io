第三次作业

第一题

1.编写Java程序声明int型变量，将int型数据转换成double类型后，进除法运算并打印结果。

上传程序截图、运行结果截图。

我的代码部分

public class yume {

public static void main(String[] args) {

// TODO Auto-generated method stub

int xjm=3;

double yzh=xjm;//这里将xjm赋值给yzh以后，yzh的值是3.0完成题目条件

yzh/=2;

System.out.println(yzh);

}

输出结果1.5。

有同学私聊我，他写的程序（部分）

int a=5;
double b=a;
b=a/2;

然后问我这个结果为啥是2.0？
当然，这是初学者很容易犯的错误，在这里我解释一下，其实它和我们上学期学的c++是一样的。
先定义了一个整型a，给a赋值5；又定义一个double型的b，把a的值赋值给b。此时b的值就是5.0。
但是这里注意了，初始化的b是5.0，后面执行的语句b=a/2;是将a/2的值赋值给b。而a是整型5，除以2结果是2，所以最后输出结果是2.0。所以很明显，这位同学的问题就在于他思考这个程序的时候逻辑有些欠缺。当然，也是大家的共性问题。编程思想同时也是以后我们做项目，打比赛最需要的地方。

第二题

编写Java程序，声明成员变量age和局部变量score，比较两个变量的区别。
上传程序截图、运行结果截图。

我设计的程序代码

package akihabara;

public class yume {
public static int  age=5;
    public static void main(String[] args) {
        // TODO Auto-generated method stub
        int score=1;
        for(int i=0;i<age;++i) {
            score*=3;
            age-=1;
        }
        int i=0;
        System.out.println(score+" "+i+" "+age);
        test();
    }
   public static void test()
   {
       System.out.println("测试:"+age);
       //如果是System.out.println("测试"+score)或者System.out.println("测试"+score)会报错;
   }
}

（1）这里成员变量 age是独立于方法之外的变量，可以在整个类中使用，而score是局部变量只能在类的方法中使用。

（2）也可以看到声明的位置也不一样，成员变量在main外面。

（3）为了更清楚地看到区别，我又写了一个for语句和一个test函数，其中的i和score是局部变量，i在for语句执行结束后就不再有任何作用了，不声明i无法使用，但是score和age可以使用。但是在test里，i和age都无法使用，如果使用就会报错。

（4）以生活来举个例子，成员变量和局部变量的关系。age是一个类的主管，在这个类里的任何方法里都是可以管到的，然而这里的局部变量score是这个类在这个方法的主管，他的手下i，只能在for语句中使用。否则需要重新声明。就是这样的区别。

第三题

编写Java程序，声明两个变量并计算两个数的乘积。

上传程序截图、运行结果截图。

我的代码部分

package akihabara;

import java.util.Scanner;

public class yume {

    public static void main(String[] args) {
        // TODO Auto-generated method stub
        Scanner input = new Scanner(System.in);

        System.out.print("Enter a number for raidus:");

        double radius = input.nextDouble();

        double area = radius * radius * 3.1415926;

        System.out.println("The area for the circle of raidus" + radius + " is " + area);
    }
}

我敲了一个圆面积的计算公式，顺便把如何输入输出的问题解决了。
